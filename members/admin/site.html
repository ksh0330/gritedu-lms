사이트 공개 콘텐츠와 기본 설정을 관리합니다.

학원 안내 페이지의 배너, 대표 인사말, 대표 사진, 하단 GRIT 문구를 관리합니다.

상단 배너

공개 페이지 최상단에 표시되는 단체 배너 이미지입니다.

assets.gritedu.kr/public/story/ 경로의 jpg, png, webp URL을 입력하세요.

대표 인사말

제목, 부제, 본문, 서명이 배너 아래 메인 텍스트 영역에 표시됩니다.

줄바꿈은 공개 페이지에 그대로 반영됩니다.

대표 사진

대표 인사말 오른쪽(모바일에서는 본문 아래)에 표시됩니다.

assets.gritedu.kr/public/story/ 경로의 jpg, png, webp URL을 입력하세요.

하단 GRIT 문구

대표 인사말 아래 G/R/I/T 설명과 마지막 강조 문구입니다.

G Growth
R Resilience
I Integrity
T Tenacity

저장 버튼을 눌러야 공개 페이지에 반영됩니다.

카카오톡 채널

공개 페이지 우측 하단에 표시할 카카오톡 채널 링크 버튼을 관리합니다.

https://pf.kakao.com/... 형식만 입력하세요. 채팅으로 바로 연결하려면 /chat이 포함된 URL을 입력할 수 있습니다.

비워두면 공개 페이지 버튼은 SVG 아이콘만 표시됩니다.

카카오톡 SDK를 사용하지 않고 단순 링크로 연결됩니다.

사업자 인증, 월렛, 매장 연결과는 별개로 동작합니다.
